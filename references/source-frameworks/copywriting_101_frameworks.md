# Copywriting 101 — Framework Summary

**Book by Brian Clark / Copyblogger (Rainmaker Digital, 2016)**

Foundational copywriting principles.

## Key Frameworks

### 1. Sugarman's Escalator Principle
Every element's sole purpose = get the NEXT element read.
Headline → first sentence → second sentence → offer → close.
Nothing exists for itself.

### 2. Features vs Benefits
Feature = what it does. Benefit = what it does FOR YOU.
Always translate features into benefits.

### 3. The Reason Why Headline
Incorporate "reasons why" in headlines. Creates curiosity + promises structure.

### 4. Irresistible Offer = Story
Frame the offer as a narrative, not a transaction.

### 5. Return Premium / Guarantee
Strong guarantee = buyers keep benefit even if they return product. Reduces perceived risk.

### 6. Storytelling Structure
Email series, landing page, or ad structured as a story with: character → problem → solution → transformation.

## General application
- Every asset should move cleanly: headline → first sentence → useful point → CTA
- "3 Reasons Your Trial Users Drop Off Early" = reason-why headline
- Guarantee message: "If any medication is substandard, we replace it free"
