# Engagement is the New Cocaine — Framework Summary

**Book on Twitter/X writing by an anonymous author (likely Mike "Wax" Janda)**

Focus: Writing for engagement on social platforms (especially Twitter/X). Useful for X strategy.

## Key Frameworks

### 1. Curiosity Gap Hook
Start with an incomplete story. Force the reader to want the next tweet/line.

### 2. Short Punch Lines
Twitter is not a blog. One thought per tweet. Short sentences.

### 3. Vulnerability > Authority
Personal stories and admissions of uncertainty outperform polished expertise.

### 4. Specificity Is Everything
Vague tweets get ignored. Named places, times, numbers, people.

### 5. Pattern Interrupt
Break expected format. Use questions, quotes, or unexpected line breaks.

### 6. The "F**k It" Principle
Write like you don't care about the algorithm. Authenticity drives engagement.

## General application
- X thread hooks: Start with a specific patient moment
- Vulnerability: "I did not understand the workflow either until I mapped it step by step"
- Specificity: "Tuesday at 3pm, the handoff stalled" beats "the process was slow"
